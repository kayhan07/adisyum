# ADİSYUM Windows Installer Branding Guide

## Brand assets

Place these assets in `deploy/windows/assets/`:

- `installer-icon.ico`
- `installer-banner.bmp`
- `tray-icon.ico`
- `splash.png`

## Installer polish

Recommended Inno Setup settings:

- custom `SetupIconFile`
- custom wizard images
- branded welcome text
- branded finish page
- version-aware splash copy

## Tray polish

Recommended tray improvements:

- custom 16x16 and 32x32 icons
- error / warning / healthy variants
- release channel badge in context menu
- branded balloon notifications

## Release UI copy

Use consistent labels:

- Adisyum POS
- Adisyum Admin
- Adisyum Desktop Bridge
- Adisyum Tray

## Brand tone

- premium
- clear
- operational
- restaurant-ready
- low-friction
